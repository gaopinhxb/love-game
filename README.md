# 那些年：爱而不得

一场关于青春、错过与重逢的互动叙事游戏。

## 简介

> 有些人教会你温柔，有些人教会你告别。而你——教会我成为自己。

这是一款基于文本的互动叙事游戏，采用分支剧情引擎驱动。你的每一个选择都会影响故事走向，最终抵达不同的结局。

## 特性

- 🎭 **性别选择**：开局选择男生/女生视角，开启截然不同的故事线
- 💕 **多角色剧情**：邂逅不同的角色，体验各自的爱情故事
- 🎯 **99种结局**：包含 HE、BE、NE、DE 等多种结局类型
- 📱 **移动端适配**：手机也可流畅游玩
- 📦 **单HTML文件**：无需服务器，打开即玩

## 玩法

1. 点击「开始故事」
2. 选择你的视角（男生/女生）
3. 阅读剧情，在关键节点做出选择
4. 你的「心动值」会随选择变化，影响剧情走向
5. 抵达属于你的结局
